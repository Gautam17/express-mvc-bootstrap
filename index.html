express-mvc-bootstrap by cliftonc

Application template for a structured Node.js, Express, Mongoose, EJS & JQuery Application

How to get started ...

Simple tutorial to get up and running with the Express-mvc-bootstrap. Tested on Ubuntu 10.10 and 10.04.

Installing Node 0.4.2

Ensure that you are logged in as the user you want to run node as (e.g. a good idea to create a www-data or node user if you are on a server and want to keep it separate from other processes.) Instructions copied from (https://github.com/joyent/node/wiki/Installation). 
cd /var/www
wget http://nodejs.org/dist/node-v0.4.2.tar.gz
tar -zxvf node-v0.4.2.tar.gz
cd node-v-0.4.2
export JOBS=2 # optional, sets number of parallel commands.
mkdir ~/local
./configure --prefix=$HOME/local/node
make
make install
export PATH=$HOME/local/node/bin:$PATH
You should now be able to type: 
cliftonc@localhost:/var/www/node-v0.4.2$ node --version
v0.4.2

Installing NPM

You can install NPM using the following commands: 
curl http://npmjs.org/install.sh | sh
And then check it via: 
npm -version
0.3.15

Installing Everything Else

npm install express
npm install mongoose
npm install ejs

Installing Bootstrap

cd /var/www
git clone git@github.com:cliftonc/express-mvc-bootstrap.git
cd express-mvc-bootstrap
./bootstrap server

Creating another model, view, controller ...

		
./bootstrap script create-model Group
./bootstrap script create-controller Group
./bootstrap script create-view Group
./bootstrap server
Or the fast way: 
		
./bootstrap script generate-all Group
./bootstrap server
